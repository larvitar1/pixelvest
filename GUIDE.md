## 1. วิธีเปิดเว็บ

เปิด File Explorer แล้วไปที่โฟลเดอร์ `Web` บน Desktop
ดับเบิลคลิกที่ไฟล์ **`index.html`** — เว็บจะเปิดขึ้นในเบราว์เซอร์ทันที

ไม่ต้องติดตั้งอะไรเพิ่ม ไม่ต้องใช้อินเตอร์เน็ต (ยกเว้นฟอนต์จาก Google ที่อาจโหลดช้าถ้าไม่มีเน็ต)

## 2. ฟีเจอร์ต่างๆ ในเว็บ

### หน้าแรก (Home)
- **ข่าวเด่น** — บทความใหญ่ที่แสดงอยู่บนสุด
- **หุ้นที่ติดตาม** — กล่องด้านขวา แสดงราคาหุ้น 6 ตัวพร้อมกราฟ
- **ข่าวล่าสุด** — กริดบทความด้านล่าง
- **เรื่องยอดนิยม** — sidebar อันดับ 1-5

### หน้าข่าวหุ้น (ข่าวหุ้น)
คลิกเมนู "ข่าวหุ้น" ด้านบน จะเห็นบทความทั้งหมดเรียงตามวันที่

### หน้าบทความ
คลิกบทความใดก็ได้เพื่ออ่านรายละเอียด มีกราฟราคาหุ้นที่เกี่ยวข้อง และบทความแนะนำด้านล่าง

### ปุ่มเฟือง (มุมล่างขวา)
ใช้เปลี่ยนได้ 3 อย่าง:
- **ธีม**: Warm (โทนเหลือง), Modern (โทนเขียว), Mono (โทนเทา — ค่าเริ่มต้น)
- **ขนาดตัวอักษร**: เลื่อน slider ซ้าย-ขวา
- **แถบราคาตลาด**: เปิด/ปิดแถบวิ่งด้านบน

## 3. วิธีแก้ไขข้อมูลหุ้น

ข้อมูลหุ้นอยู่ที่ไฟล์ **`js/data.js`** เปิดด้วย Notepad หรือโปรแกรมแก้ข้อความใดก็ได้

### ตัวอย่าง: เปลี่ยนราคาและ % การเปลี่ยนแปลง

ค้นหาบรรทัดที่มีชื่อหุ้นที่ต้องการ เช่น NVDA:

```
{ sym:'NVDA', name:'NVIDIA Corp.', ... price:131.50, pct:2.94, ... }
```

แก้แค่ 2 ค่านี้:
- **`price`** — ราคาหุ้น (ตัวเลข)
- **`pct`** — % เปลี่ยนแปลง (ถ้าติดลบใส่ `-` ข้างหน้า เช่น `-1.5`)

**ตัวอย่าง:** อยากให้ NVDA ราคา 145.00 บวก 1.5%
```
price:145.00, pct:1.5,
```

### เพิ่มหุ้นใหม่

คัดลอกบรรทัดหุ้นใดก็ได้ แล้ววางต่อท้ายก่อนบรรทัด `];` และแก้ค่า:

```js
{ sym:'TSM',  name:'Taiwan Semiconductor', sector:'เซมิคอนดักเตอร์', exchange:'NYSE', price:180.00, pct:0.85, sd:99 },
```

> **หมายเหตุ:** `sd` คือตัวเลขสุ่มสำหรับสร้างกราฟ ใส่ตัวเลขใดก็ได้ที่ยังไม่ซ้ำกับตัวอื่น

### แก้ดัชนีตลาด (S&P 500, NASDAQ, DOW)

ในไฟล์เดียวกัน ค้นหาส่วน `INDICES`:

```js
const INDICES = [
  { name:'S&P 500',    value:5468.55,  chg:24.32  },
  ...
];
```

แก้ **`value`** (ค่าดัชนี) และ **`chg`** (การเปลี่ยนแปลง ลบได้)

## 4. วิธีเพิ่ม/แก้ไขบทความข่าว

บทความอยู่ที่ **`js/data.js`** ในส่วน `const ARTS = [...]`

### โครงสร้างบทความ 1 ชิ้น

```js
{ 
  id: 9,                          // เลขไม่ซ้ำ (ถ้าเพิ่มใหม่ใส่เลขต่อจาก 8)
  cat: 'เทคโนโลยี',               // หมวดหมู่ (แสดงเป็น tag)
  title: 'หัวข้อข่าว',            // หัวข้อ
  excerpt: 'สรุปย่อ...',          // ข้อความสั้นๆ ใต้หัวข้อ
  author: 'ทีมข่าว PixelVest',    // ชื่อผู้เขียน
  date: '28 มิ.ย. 2026',          // วันที่
  read: 3,                        // เวลาอ่านโดยประมาณ (นาที)
  syms: ['AAPL'],                 // สัญลักษณ์หุ้นที่เกี่ยวข้อง
  rank: 9,                        // อันดับความนิยม (ใช้กับ sidebar)
  body: {
    intro: 'ย่อหน้าเปิดบทความ...',
    sections: [
      { h: 'หัวข้อย่อย 1', p: 'เนื้อหาย่อหน้าแรก...' },
      { h: 'หัวข้อย่อย 2', p: 'เนื้อหาย่อหน้าสอง...' },
    ],
    quote: 'ประโยคเด่นที่อยากไฮไลต์...',
    quoteBy: '— ที่มาของคำพูด',
    takeaway: 'สรุปสิ่งที่ควรจำจากบทความ...',
  }
},
```

### เพิ่มบทความใหม่

1. เปิด `js/data.js`
2. ไปที่บรรทัดสุดท้ายก่อน `];` ของ `ARTS`
3. เพิ่ม `,` ต่อท้ายบทความสุดท้าย แล้ววางโครงสร้างข้างบน
4. บันทึกไฟล์ แล้วรีเฟรชเบราว์เซอร์

### แก้บทความที่มีอยู่

ค้นหาชื่อบทความด้วย Ctrl+F แล้วแก้ค่าที่ต้องการ บันทึก แล้วรีเฟรช

## 5. วิธีบันทึกการแก้ไข (Git)

Git คือระบบบันทึกประวัติการแก้ไข เหมือน "จุด save" ในเกม — ถ้าทำอะไรพังก็ย้อนกลับได้

### วิธีบันทึก (Commit)

1. เปิด **Terminal** (กด `Win + X` แล้วเลือก Terminal)
2. พิมพ์คำสั่งทีละบรรทัด กด Enter หลังแต่ละบรรทัด:

```
cd C:\Users\ASUS\Desktop\Web
git add .
git commit -m "อธิบายว่าแก้อะไร เช่น อัปเดตราคาหุ้น NVDA"
```

### ดูประวัติ commit ทั้งหมด

```
git log --oneline
```

จะเห็นรายการเหมือนนี้:
```
968cab0 จัดโครงสร้างไฟล์ใหม่เรียบร้อย
2e9d38a ก่อนจัดโครงสร้างไฟล์ใหม่
```

## 6. วิธีย้อนกลับไปเวอร์ชันเก่า

### ย้อนกลับไฟล์เดียว

ถ้าแก้ไขไฟล์ใดแล้วพัง แต่ยังไม่ได้ commit:

```
git checkout -- js/data.js
```

(เปลี่ยน `js/data.js` เป็นชื่อไฟล์ที่ต้องการ)

### ย้อนกลับไป commit ก่อนหน้า (ทุกไฟล์)

ดูเลข commit จาก `git log --oneline` แล้วใช้:

```
git reset --hard 2e9d38a
```

(เปลี่ยน `2e9d38a` เป็นเลขของ commit ที่ต้องการ)

> **คำเตือน:** คำสั่งนี้จะลบการแก้ไขที่ยังไม่ได้ commit ทิ้งทั้งหมด

## 7. คำถามที่พบบ่อย

**ถาม: แก้ไขแล้วเว็บไม่เปลี่ยน ทำอย่างไร?**
กด `Ctrl + Shift + R` (หรือ `Ctrl + F5`) ในเบราว์เซอร์เพื่อ hard refresh ล้าง cache

**ถาม: เปิดไฟล์แล้วเห็นแต่โค้ด จะแก้ด้วยโปรแกรมไหน?**
ใช้ Notepad (ติดมากับ Windows) ได้เลย หรือโหลด [Notepad++](https://notepad-plus-plus.org/) ฟรี อ่านง่ายกว่ามาก

**ถาม: ทำไมฟอนต์ดูแปลกเมื่อเปิดโดยไม่มีเน็ต?**
ฟอนต์โหลดจาก Google Fonts ต้องการอินเตอร์เน็ต ถ้าไม่มีเน็ตจะใช้ฟอนต์ระบบแทน หน้าตายังใช้งานได้ปกติ

**ถาม: ต้องการเพิ่มรูปภาพจริงแทนกล่องสีเทา ทำได้ไหม?**
ได้ครับ ต้องสร้างโฟลเดอร์ `images/` แล้วแก้โค้ดใน `js/views.js` — แนะนำให้ขอความช่วยเหลือจาก Claude Code

**ถาม: จะ deploy ขึ้นเว็บจริงได้ไหม?**
ได้ เพราะเป็น static website ทั้งหมด อัปโหลดทุกไฟล์และโฟลเดอร์ขึ้น hosting ได้เลย เช่น GitHub Pages, Netlify (ทั้งสองฟรี)
