## 1. วิธีเปิดเว็บ

เปิด File Explorer แล้วไปที่โฟลเดอร์ `Web` บน Desktop
ดับเบิลคลิกที่ไฟล์ **`index.html`** — เว็บจะเปิดขึ้นในเบราว์เซอร์ทันที

ไม่ต้องติดตั้งอะไรเพิ่มสำหรับ "ดูเว็บ"

> **หมายเหตุ:** คำสั่งที่ต้องพิมพ์ใน Terminal (เช่น `node build.js`, `node fetch-quotes.js`)
> ต้องติดตั้ง **Node.js** ก่อน (โหลดฟรีจาก [nodejs.org](https://nodejs.org/)) ติดตั้งครั้งเดียวพอ

## 2. ฟีเจอร์ต่างๆ ในเว็บ

### หน้าแรก (Home)
- **ข่าวเด่น** — บทความใหญ่ที่แสดงอยู่บนสุด (ข่าวภาพรวมตลาดล่าสุด)
- **หุ้นที่ติดตาม** — กล่องด้านขวา แสดงราคาหุ้น 6 ตัวพร้อมกราฟ + เวลาอัปเดตล่าสุด
- **ข่าวล่าสุด** — กริดบทความด้านล่าง
- **เรื่องยอดนิยม** — sidebar อันดับ 1-5

### หน้าหุ้นรายตัว (คลิกหุ้นใน "หุ้นที่ติดตาม")
คลิกที่หุ้นตัวใดก็ได้ในกล่อง "หุ้นที่ติดตาม" (หรือชิปหุ้นในบทความ) จะเข้าหน้ารายตัว มี:
- **กราฟราคา + ปริมาณซื้อขาย** เลือกช่วงเวลาได้ `1D / 1W / 1M / 3M / 1Y`
- **ตัวเลขสำคัญ** — ราคา, % เปลี่ยนแปลง, ตลาด, กลุ่มอุตสาหกรรม
- **ผลประกอบการล่าสุด (Joy)** และ **บทวิเคราะห์พื้นฐาน (Max)** — ดึงจากบทความที่เกี่ยวกับหุ้นตัวนั้น
- **ข่าวที่เกี่ยวข้อง** กับหุ้นตัวนั้น

### หน้าข่าวหุ้น (เมนู "ข่าวหุ้น")
คลิกเมนู "ข่าวหุ้น" ด้านบน จะเห็นบทความทั้งหมดเรียงตามวันที่

### หน้าบทความ
คลิกบทความใดก็ได้เพื่ออ่านฉบับเต็ม มีป้ายระบุว่าใครเป็นผู้เรียบเรียง (Joy / Max / Hanako)
ตารางตัวเลขสำคัญ กราฟหุ้นที่เกี่ยวข้อง และแหล่งอ้างอิงท้ายบทความ

### ปุ่มเฟือง (มุมล่างขวา)
- **ธีม**: Warm (โทนเหลือง), Modern (โทนเขียว), Mono (โทนเทา — ค่าเริ่มต้น)
- **ขนาดตัวอักษร**: เลื่อน slider ซ้าย-ขวา
- **แถบราคาตลาด**: เปิด/ปิดแถบวิ่งด้านบน

## 3. อัปเดตราคาหุ้นจริง (/Update)

ราคาหุ้นและดัชนีบนเว็บดึงมาจาก **Yahoo Finance** ของจริง (ไม่ต้องสมัคร ไม่ต้องใช้รหัส)

### วิธีที่ง่ายที่สุด — สั่ง Claude
พิมพ์ในแชต Claude Code ว่า:

```
/Update
```

Claude จะดึงราคาล่าสุดให้ แล้วบอกว่าอัปเดตกี่รายการ เวลาเท่าไร

### หรือทำเองใน Terminal
```
cd C:\Users\ASUS\Desktop\Web
node fetch-quotes.js
```
แล้วรีเฟรชเบราว์เซอร์ — ราคาทุกที่ (แถบบน, หุ้นที่ติดตาม, หน้าหุ้น) จะอัปเดตตาม

### สิ่งที่ควรรู้
- ราคาเป็น **snapshot** = ค่าตอนที่กดอัปเดต ไม่ใช่ราคาวิ่งเองตลอดเวลา → อยากได้ล่าสุดก็สั่ง `/Update` ซ้ำ
- ถ้าเน็ตล่มหรือ Yahoo บล็อกชั่วคราว เว็บจะใช้ราคาเดิมต่อไป ไม่พัง
- **กราฟยังเป็นเส้นจำลอง** (ขึ้น/ลงตามทิศ % จริง แต่ยังไม่ใช่ราคาย้อนหลังจริง)

## 4. เขียนข่าวด้วย AI Agent (/Brief)

มีผู้ช่วย AI 3 คนเขียนข่าวลงเว็บให้อัตโนมัติ สั่งผ่านแชต Claude Code

| คำสั่ง | ผู้ช่วย | ได้อะไร |
|---|---|---|
| `/Brief Hanako` | Hanako | ข่าวภาพรวมตลาดวันนี้ (ขึ้นช่องข่าวเด่นหน้าแรก) |
| `/Brief Joy NVDA` | Joy | ข่าวสรุปผลประกอบการของหุ้นที่ระบุ |
| `/Brief Max NVDA` | Max | บทวิเคราะห์พื้นฐานของหุ้นที่ระบุ |

> เปลี่ยน `NVDA` เป็นหุ้นตัวอื่นได้ แต่ต้องเป็นหุ้นที่เว็บรองรับ (ดูรายชื่อในหัวข้อ 5)

ผู้ช่วยจะค้นข้อมูลจริง เขียนไฟล์ข่าว แล้วบอกให้รัน `node build.js` (Claude มักรันให้เอง)
ทุกข่าวมี **แหล่งอ้างอิง** และ **ตัวเลขสำคัญ** ติดมาด้วย

## 5. เขียน/แก้บทความเอง (Markdown)

> ⚠️ **เปลี่ยนระบบแล้ว!** บทความ **ไม่ได้อยู่ใน `js/data.js`** อีกต่อไป
> ตอนนี้ข่าว 1 ชิ้น = ไฟล์ `.md` 1 ไฟล์ ในโฟลเดอร์ **`articles/`**

### ขั้นตอน
1. ก๊อปไฟล์ **`articles/TEMPLATE.md`** เป็นไฟล์ใหม่ ตั้งชื่อขึ้นต้นด้วยเลข เช่น `11-ชื่อข่าว.md`
   (เลขหน้าไฟล์ = รหัสข่าว ห้ามซ้ำกับไฟล์อื่น)
2. แก้เนื้อหาในไฟล์
3. เปิด Terminal แล้วรัน:
   ```
   cd C:\Users\ASUS\Desktop\Web
   node build.js
   ```
4. รีเฟรชเบราว์เซอร์ → ข่าวใหม่ขึ้นทันที

### โครงไฟล์ข่าว (.md)
```markdown
---
title: หัวข้อข่าว
cat: หมวด เช่น เซมิคอนดักเตอร์
type: news          # news (ปกติ) | featured (ข่าวเด่น) | analysis (บทวิเคราะห์)
agent: joy          # ใครเขียน joy | max | hanako (ไม่ใส่ก็ได้)
author: ชื่อผู้เขียน
date: 2026-06-28     # ระบบแปลงเป็นวันไทยให้เอง
syms: [NVDA]         # หุ้นที่เกี่ยวข้อง
excerpt: สรุปสั้นบนการ์ด
---

ย่อหน้าเปิด (บทนำ)

## หัวข้อย่อย
เนื้อหา...

> ประโยคเด่น
> — ที่มา

**สรุป:** ประเด็นที่ควรจำ

## ตัวเลขสำคัญ
- รายได้ | 215.94B | +65%
- EPS | 2.94 | +71%

## แหล่งอ้างอิง
- [ชื่อบทความ — สำนักข่าว](https://example.com)
```

- `## ตัวเลขสำคัญ` → ขึ้นเป็นตาราง key stats ใต้กราฟในหน้าหุ้น (รูปแบบ `- ป้าย | ค่า | เปลี่ยนแปลง`)
- `## แหล่งอ้างอิง` → ขึ้นเป็นลิงก์ท้ายบทความ
- คำอธิบายเต็มอยู่ใน `articles/TEMPLATE.md`

> **อย่าแก้ `js/articles.js` ตรง ๆ** — ไฟล์นั้นสร้างอัตโนมัติจาก `node build.js`

## 6. แก้ไขข้อมูลหุ้นและดัชนี

ข้อมูลหุ้นอยู่ที่ **`js/data.js`** เปิดด้วย Notepad ก็ได้

> **หมายเหตุสำคัญ:** ถ้าใช้ `/Update` ดึงราคาจริงไว้ ราคาในเว็บจะใช้ค่าจาก Yahoo
> (ในไฟล์ `js/quotes.js`) ทับค่าใน `data.js` — ค่าใน `data.js` เป็นแค่ค่าสำรองเมื่อยังไม่มีราคาจริง

### เพิ่มหุ้นใหม่เข้าระบบ
หุ้นที่เว็บรองรับตอนนี้มี 10 ตัว:
**NVDA AAPL MSFT AMZN GOOGL META TSLA AMD JPM NFLX**

อยากเขียนข่าวหุ้นนอกรายชื่อนี้ ต้องเพิ่มเข้า `STOCKS` ก่อน (ไม่งั้นกราฟ/ชิปหุ้นจะเพี้ยน)
คัดลอกบรรทัดหุ้นใดก็ได้ วางต่อท้ายก่อน `];` แล้วแก้ค่า:
```js
{ sym:'TSM', name:'Taiwan Semiconductor', sector:'เซมิคอนดักเตอร์', exchange:'NYSE', price:180.00, pct:0.85, sd:99 },
```
> `sd` = ตัวเลขสุ่มสำหรับสร้างกราฟ ใส่เลขใดก็ได้ที่ยังไม่ซ้ำ
> เพิ่มหุ้นใหม่แล้วอย่าลืมเพิ่ม ticker เข้าใน `fetch-quotes.js` ด้วย ถ้าอยากให้ `/Update` ดึงราคาตัวนั้น

### แก้ดัชนีตลาด (S&P 500, NASDAQ, DOW)
ในไฟล์เดียวกัน ค้นหาส่วน `INDICES` แก้ `value` (ค่าดัชนี) และ `chg` (การเปลี่ยนแปลง)
— แต่ถ้าใช้ `/Update` ดัชนีก็จะถูกอัปเดตเป็นค่าจริงให้เองอยู่แล้ว

## 7. สรุปคำสั่งลัดทั้งหมด

พิมพ์ในแชต Claude Code:

| คำสั่ง | ทำอะไร |
|---|---|
| `/Update` | ดึงราคาหุ้น/ดัชนีล่าสุดจาก Yahoo |
| `/Brief Hanako` | เขียนข่าวภาพรวมตลาดวันนี้ |
| `/Brief Joy <หุ้น>` | เขียนข่าวผลประกอบการ |
| `/Brief Max <หุ้น>` | เขียนบทวิเคราะห์พื้นฐาน |

คำสั่งใน Terminal:

| คำสั่ง | ทำอะไร |
|---|---|
| `node fetch-quotes.js` | ดึงราคาจริง (เบื้องหลังของ `/Update`) |
| `node build.js` | แปลงไฟล์ข่าว `.md` เป็นเว็บ |

## 8. บันทึกการแก้ไข (Git)

Git คือระบบบันทึกประวัติการแก้ไข เหมือน "จุด save" ในเกม — ถ้าทำอะไรพังก็ย้อนกลับได้

### วิธีบันทึก (Commit)
1. เปิด **Terminal** (กด `Win + X` แล้วเลือก Terminal)
2. พิมพ์ทีละบรรทัด กด Enter หลังแต่ละบรรทัด:
```
cd C:\Users\ASUS\Desktop\Web
git add .
git commit -m "อธิบายว่าแก้อะไร เช่น เพิ่มข่าวผลประกอบการ NVDA"
```

### ดูประวัติ commit
```
git log --oneline
```

## 9. ย้อนกลับไปเวอร์ชันเก่า

### ย้อนกลับไฟล์เดียว (ยังไม่ได้ commit)
```
git checkout -- js/data.js
```
(เปลี่ยนเป็นชื่อไฟล์ที่ต้องการ)

### ย้อนกลับไป commit ก่อนหน้า
ดูเลข commit จาก `git log --oneline` แล้ว:
```
git reset --hard 2e9d38a
```
(เปลี่ยน `2e9d38a` เป็นเลขของ commit ที่ต้องการ)

> **คำเตือน:** คำสั่งนี้จะลบการแก้ไขที่ยังไม่ได้ commit ทิ้งทั้งหมด
> หรือบอก Claude เป็นภาษาไทยว่า "ย้อนกลับ" ก็ได้ — Claude จะเลือกวิธีที่ปลอดภัยให้

## 10. คำถามที่พบบ่อย

**ถาม: แก้ไขแล้วเว็บไม่เปลี่ยน ทำอย่างไร?**
กด `Ctrl + Shift + R` (หรือ `Ctrl + F5`) ในเบราว์เซอร์เพื่อ hard refresh ล้าง cache

**ถาม: เขียนข่าว `.md` แล้ว แต่ไม่ขึ้นในเว็บ?**
ลืมรัน `node build.js` หรือเปล่า — ทุกครั้งที่แก้ไฟล์ใน `articles/` ต้องรัน `node build.js` ใหม่

**ถาม: `/Update` แล้วราคาบางตัวไม่อัปเดต?**
Yahoo อาจดึงตัวนั้นไม่ได้ชั่วคราว ตัวที่เหลือยังอัปเดตปกติ ลองสั่ง `/Update` ใหม่อีกครั้ง

**ถาม: พิมพ์ `node` แล้วขึ้นว่าไม่รู้จักคำสั่ง?**
ยังไม่ได้ติดตั้ง Node.js — โหลดฟรีจาก [nodejs.org](https://nodejs.org/) ติดตั้งแล้วเปิด Terminal ใหม่

**ถาม: เปิดไฟล์แล้วเห็นแต่โค้ด จะแก้ด้วยโปรแกรมไหน?**
ใช้ Notepad (ติดมากับ Windows) ได้เลย หรือโหลด [Notepad++](https://notepad-plus-plus.org/) ฟรี อ่านง่ายกว่ามาก

**ถาม: จะ deploy ขึ้นเว็บจริงได้ไหม?**
ได้ เพราะเป็น static website อัปโหลดทุกไฟล์ขึ้น GitHub Pages หรือ Netlify (ฟรี) ได้เลย
