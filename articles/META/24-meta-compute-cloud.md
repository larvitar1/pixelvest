Meta Platforms ประกาศแผนเปิดตัวหน่วยธุรกิจใหม่ชื่อ **Meta Compute** เพื่อขายกำลังประมวลผล AI ส่วนเกินให้บุคคลภายนอก ท้าทาย AWS, Azure และ Google Cloud โดยตรง ข่าวนี้ดันหุ้น META พุ่งขึ้น 10.34% ในวันที่ 1 กรกฎาคม 2026 เพราะช่วยเปลี่ยน narrative จาก "เผาเงิน CapEx" เป็น "สร้างรายได้ใหม่"

## จาก cost center สู่ revenue line

Meta ลงทุนสร้างโครงสร้างพื้นฐาน AI มหาศาล โดย CapEx ปี 2026 อยู่ในกรอบ 125–145 พันล้านดอลลาร์ เกือบสองเท่าของปี 2025 ที่ 69.69 พันล้าน ตลาดกังวลมาตลอดว่าการลงทุนระดับนี้จะคุ้มค่าเมื่อไหร่ แต่ Meta Compute เสนอคำตอบที่ชัดเจน — แทนที่จะปล่อยให้กำลังประมวลผลว่างเปล่า บริษัทจะขายมันออกไปเป็นบริการ cloud

รูปแบบธุรกิจคล้ายกับ neocloud อย่าง CoreWeave คือสามารถขาย hardware access โดยตรง หรือ host โมเดล AI ให้นักพัฒนาเรียกใช้ผ่าน API ได้ โดย Meta มีข้อได้เปรียบจาก data center 30 แห่ง (26 แห่งในสหรัฐฯ) ที่กำลังก่อสร้างภายใต้แผนลงทุน 600 พันล้านดอลลาร์ภายในปี 2028

## ทำไมตลาดถึงตื่นเต้น

เหตุผลที่หุ้นพุ่ง 10% ในวันเดียวเป็นเรื่องของ perception shift มากกว่าตัวเลข เพราะยังไม่มีรายละเอียดรายได้จริง แต่การประกาศ Meta Compute ส่งสัญญาณว่า:

- CapEx มหาศาลไม่ใช่ sunk cost แต่เป็นสินทรัพย์ที่สร้างรายได้ได้
- Meta จะมี revenue stream ใหม่นอกเหนือจากโฆษณาเป็นครั้งแรกอย่างแท้จริง
- บริษัทเข้าสู่ตลาด cloud ที่มีมูลค่ารวมกว่า 600 พันล้านดอลลาร์ต่อปี

## ความท้าทายที่ยังต้องพิสูจน์

แม้ข่าวจะดี แต่ Meta เข้าตลาด cloud ช้ากว่าคู่แข่งหลายปี AWS, Azure และ Google Cloud มี ecosystem ที่แข็งแกร่ง มีลูกค้าองค์กรจำนวนมาก และมี managed services ครบวงจร Meta Compute จะต้องแข่งด้วยราคาและ AI-native infrastructure เป็นหลัก ซึ่งใกล้เคียงกับรูปแบบของ CoreWeave ที่เน้น GPU-as-a-service มากกว่า full-stack cloud

นอกจากนี้ยังต้องดูว่าจะมี conflict of interest กับลูกค้าโฆษณารายใหญ่ที่ใช้ cloud ของคู่แข่งอยู่แล้วหรือไม่

> การเข้าสู่ธุรกิจ cloud ของ Meta ไม่ใช่แค่การหารายได้เพิ่ม แต่เป็นการพิสูจน์ให้ตลาดเห็นว่า CapEx ระดับแสนล้านดอลลาร์ต่อปี สามารถแปลงเป็นสินทรัพย์ที่ทำเงินได้จริง
> — ทีมข่าว PixelVest

**สรุป:** Meta Compute เป็น catalyst สำคัญที่ช่วยคลายความกังวลเรื่อง CapEx ของ Meta ได้ในระดับหนึ่ง แม้ยังต้องพิสูจน์ว่าจะแข่งกับ hyperscaler รายใหญ่ได้จริงหรือไม่ แต่การที่หุ้นพุ่ง 10% ในวันเดียว สะท้อนว่าตลาดต้องการเห็นทางออกของ CapEx มหาศาลนี้มานานแล้ว
